# VHDL Learning Projects

Repositorio para práctica personal de diseño digital usando VHDL.
Incluye módulos simples (AND, OR, MUX, FFs), testbenches y scripts de simulación.

## Estructura
- modules/  -> cada módulo con su testbench
- scripts/  -> comandos automatizados
- docs/     -> notas y diagramas

## Requisitos
- GHDL
- GTKWave
